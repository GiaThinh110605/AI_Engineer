# Docker
### Lý do: đóng gói project(thư viện, hệ điều hành, data...) để chạy trên máy khách
### Chỉ cần: mở ra trên máy khách là chạy
- Docker hub: là cộng đồng đã làm sẵn những docker image, mình có thể kế thừa ở trong docker
- Docker file: định nghĩa các docker image
- Docker image ~ class: định nghĩa những thứ cần để chạy được chương trình của bạn
- Docker container ~ object: có thể chạy được nhiều docker container từ 1 image
